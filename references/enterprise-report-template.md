# Enterprise SQL Report Template

Use this structure for substantial SQL design, review, or optimization deliverables.

## Optimization Report

```text
Conclusion
Confirmed facts
Missing evidence and assumptions
Current SQL or code path
Current schema and indexes
Execution plan observations
Root cause
Recommended change
Alternative options
Security findings
Index impact
Locking and transaction impact
Version compatibility
Validation plan
Deployment plan
Rollback plan
Open questions
```

## Design Report

```text
Design summary
Business assumptions
Database selection
Schema
Constraints
Indexes
SQL or ORM implementation
Security model
Transaction and concurrency model
Migration plan
Testing plan
Operational notes
Open questions
```
